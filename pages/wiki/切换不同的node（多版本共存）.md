#### 安装：（win）
* `git clone git://github.com/hakobera/nvmw.git1`
* 将此文件夹加入到Path环境变量中。（注意需要把路径的最后一个/加入，否则install时会发生错误）

#### 使用：
* `nvmw install v0.11.9` #下载v0.11.9的版本
* `nvmw ls` #查看当前node版本
* `nvmw use v0.11.9` #设定当前node版本
